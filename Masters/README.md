# /Masters
This folder contains the original, unsplit master files that were used to create all of the file conversions in the top level of the repository. The language data for each language in each of these files was split off and converted into its own file in the same format. For example, the Arabic data in the `Arabic_ara.conllu` file was originally contained in `CONLLUmaster.conllu`. This data was simply split off and put into its own file. 

These can be used to visualize how language data in differing languages appear in one format.
